# 错题根因分类

| 类别 | 识别信号 | 修复动作 |
|---|---|---|
| 拓扑识别 | 漏支路、跨线误并、串并联判断错 | 重建节点表与支路表，重绘核验 |
| 参考方向 | KCL/KVL 符号混乱、负值解释错 | 在列方程前标全电压极性和电流箭头 |
| 模型选择 | 稳态/瞬态、时域/相量、线性/非线性模型混用 | 写出方法适用条件和电路所处状态 |
| 初始条件 | 电容电压或电感电流换路前后处理错 | 先求 `0-`，应用连续性，再得到 `0+` |
| 定理条件 | 受控源置零、叠加功率、最大功率条件误用 | 单列定理条件并逐项核对 |
| 方程建立 | 节点、回路或相量方程缺项 | 用节点支路度数或独立方程数检查 |
| 代数计算 | 矩阵、复数、指数或单位换算错误 | 保留符号到最后，回代原方程 |
| 结果表达 | 缺单位、方向、相角范围或物理含义 | 使用“数值 + 单位 + 参考方向解释” |
| 时间管理 | 方法过长、重复计算、无自检 | 先识别标准模型，预留最后 10% 时间检查 |

复盘时选择最早导致错误的主类别，可补充一个次要类别。不要为凑表格给一次错误贴多个模糊标签。
