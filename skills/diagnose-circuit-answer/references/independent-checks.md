# 独立复验方法

主解和复验必须尽量避免共同失效模式。

| 主解类型 | 优先复验 | 不足以独立复验的做法 |
|---|---|---|
| 节点电位法 | 回路法、支路法、戴维南/诺顿、KCL 回代 | 把同一节点方程再抄一遍 |
| 回路电流法 | 节点法、KVL 回代、功率守恒 | 仅重新解同一矩阵 |
| 戴维南/诺顿 | 端口测试源、开短路结果、直接列方程 | 只对最终分数重新按计算器 |
| 叠加定理 | 原电路直接列方程、齐次性/极限检查 | 重新相加同一组分响应 |
| 一阶三要素法 | 微分方程回代、初终值和时间常数物理检查 | 只检查指数形式长得像 |
| 拉普拉斯法 | 时域初终值、微分方程回代 | 只做相同部分分式展开 |
| 相量法 | KCL/KVL 复数回代、功率平衡、极限频率 | 只比较相角格式 |
| 谐振/频响 | 低频高频极限、半功率点、量纲 | 只重新代入同一公式 |
| 三相电路 | 每相功率与总功率关系、线相量关系 | 仅重复根号 3 运算 |
| 非线性/二极管 | 分段假设一致性、工作点回代 | 只接受最初导通假设 |

## 通用快速检查

- **方程回代**：把最终量代回所有独立 KCL/KVL，而不是只代回最后一个式子。
- **量纲**：等号两边单位一致；相量、有效值、峰值没有混用。
- **符号**：负值与预设参考方向的物理解释一致。
- **功率守恒**：采用统一参考约定后，所有元件功率代数和为零。
- **初终值**：电容电压与电感电流满足换路连续性；终值符合直流稳态等效。
- **极限**：令参数趋于 0、无穷或频率极限，结果符合开路、短路或退化电路。
- **数量级**：电压、电流、时间常数、功率在元件和电源约束下合理。

报告实际检查式与残差。只写“已用 KCL 检查”不算复验。
